# 手迹造字 WriteFont

从手写样本到可安装字体，全链路自动化。Android 原生应用，拍照即出字。

## 📥 下载安装

**最新版本：v4.3.0** | [全部版本](https://github.com/MrLing1202/writefont/releases)

| 文件名 | 适用设备 |
|--------|----------|
| `app-arm64-v8a-release.apk` | **主流手机选这个**（推荐） |
| `app-armeabi-v7a-release.apk` | 老款 32 位手机 |
| `app-x86_64-release.apk` | 安卓模拟器 |

> 下载 → 允许安装未知来源 → 安装完成

## ✨ 核心功能

### 📸 拍照造字
- **智能拍照识别** — 对准手写方格纸拍照，AI 自动识别每个字符
- **双引擎投票** — ML Kit + TFLite 两个 AI 引擎独立识别，投票选出最优结果
- **多级预处理投票** — 25+ 种图像预处理策略（含形态学断笔修复/细笔画增强）× 多级放大，投票选出最佳识别结果
- **自适应提前终止** — 根据图片大小动态调整阈值，小图更快，大图更准
- **形近字区分** — 内置 350+ 组形近字对照表，覆盖手写体高频混淆
- **同音字纠错** — 300+ 组同音字 + 500+ 常用词库 + 500+ 三字短语/成语，2-4 字上下文感知自动纠错
- **字典后处理** — 常见字优先，过滤生僻误识别
- **增强置信度校准** — 共识强度、TFLite 一致性、字典验证多维度校准

### 🔤 字体生成
- **InkForge 引擎** — 自研 TTF 生成引擎，贝塞尔曲线拟合，亚像素级轮廓精度
- **一键生成 TTF** — 3500 字字体 < 3 分钟
- **字体家族生成** — 从一套手写生成 Regular/Bold/Light 多重变体
- **OTF 导出** — 支持 OpenType 格式
- **Google Fonts 格式** — 可直接用于网页嵌入

### ✏️ 字符编辑
- **字符编辑器** — 缩放、旋转、裁剪、笔画粗细调节
- **撤销/重做** — 完整操作历史，支持 50 步回退
- **批量照片导入** — 一次导入多张手写照片，批量识别
- **自定义文本预览** — 输入任意文本实时预览字体效果

### 📊 分析工具
- **字符集完整性分析** — 检查常用字覆盖率，找出缺失字符
- **字形自动补全** — 基于已有字形风格，AI 推断缺失字符
- **字体对比工具** — 并排对比不同字体差异
- **字形评分系统** — 自动评估字形质量
- **Kerning 编辑器** — 调整字符间距
- **风格分析器** — 分析字体风格特征

### 🎯 识别率优化（本地离线）
- **用户反馈学习** — 纠正过的字符自动记住，下次识别更准
- **多候选 Top-3** — 识别不确定时给出多个候选，用户选择
- **图像质量评估** — 自动检测模糊、低对比度、噪点并增强
- **手写体专用预处理** — 笔画增强、倾斜校正、笔画归一化
- **CLAHE 对比度自适应** — 局部自适应直方图均衡，浅色笔迹也能识别
- **USM 多尺度笔画锐化** — 多尺度 Unsharp Masking 锐化，模糊图片自动增强笔画清晰度
- **智能白边裁剪** — 自动检测并裁剪图片四周空白，让字符占据更大比例，提升识别准确率
- **笔画粗细自适应** — 自动检测笔画宽度，过细膨胀、过粗腐蚀，归一化到最佳识别范围
- **TFLite 本地手写识别** — 内置轻量级手写识别模型，无需联网，与 ML Kit 双引擎投票提升准确率
- **同音字自动纠错** — 利用上下文词组关系，自动纠正"读对但选错"的同音字（如"在/再"、"的/地/得"）
- **粘连字符分割** — 垂直投影法自动检测并切分粘连字符，连笔也能正确识别
- **形态学断笔修复** — 闭运算填充笔画间隙，开运算去噪，细笔画自适应增强
- **策略权重学习** — 预处理策略成功率持久化保存，越用越准，每天自动衰减防止过拟合
- **异常检测过滤** — 自动过滤标点、数字、字母等非汉字结果，识别更干净

### 🌐 其他
- **网页嵌入** — 生成 Web Font 可直接嵌入网页
- **深色模式** — 跟随系统或手动切换
- **多语言** — 中文 / English / 日本語

## 📐 使用流程

1. 方格纸手写常用汉字（建议 3500 字）
2. 打开 WriteFont，拍照导入
3. AI 自动识别 → 确认/纠正字符
4. 生成 TTF 字体 → 安装到手机或导出

## 📝 注意事项

- 方格纸网格越清晰，识别越准
- 拍照保持纸张平整、光线均匀
- 纠正过的字符会被记住，越用越准
- 生成的 TTF 全平台通用（Windows/macOS/Linux/Android/iOS）
