# Release Notes

## v4.0.0 (2026-06-23)

### ✏️ 识别更准 — 模糊字迹也能看清

- **USM 多尺度笔画锐化**：全新锐化引擎，拍照时轻微抖动或墨迹较浅的字迹也能自动增强笔画清晰度
- **智能模糊检测**：系统自动判断图片是否模糊，模糊时自动触发增强，无需手动操作
- **三种锐化强度**：轻度锐化、标准锐化、锐化+对比度增强组合，覆盖各种模糊场景

### 适用场景
- 拍照时手抖导致的模糊
- 墨迹较浅、笔画不清晰的手写
- 复印件或扫描件中的模糊字迹
- 光线不佳导致的细节丢失

## v3.9.0 (2026-06-23)

### 🎯 识别更准 — 浅色笔迹也能识别

- **对比度自适应增强**：拍照识别时自动检测图片明暗，对浅色墨水、铅笔书写、光线不足的照片进行智能增强
- **四档自动调节**：根据图片对比度自动选择最佳增强强度，正常图片不处理避免过度增强
- **识别策略升级**：快速通道和完整识别管线均接入自适应增强，整体识别准确率提升

### 适用场景
- 浅色铅笔、圆珠笔书写
- 光线不足环境下拍照
- 复印件、扫描件中的浅色字迹
- 纸张泛黄或墨迹褪色的手写
