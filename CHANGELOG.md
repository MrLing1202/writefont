# Changelog

所有重要更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)。

## [2.6.0] - 2026-06-22

### 新增
- 4种新预处理策略：自适应直方图均衡（四象限法）、形态学骨架化、高斯模糊去噪+锐化、局部阈值二值化
- 策略可靠性追踪：历史成功率用于加权投票（高可靠性策略 1.2x 权重）
- 智能投票增强：票数接近时执行平局决胜、置信度校准（投票边际+策略多样性）
- 总计 19 种预处理策略参与投票

### 修复
- 高斯核计算：替换自实现泰勒级数 exp() 为 dart:math 标准库 exp()，精度提升

## [2.5.0] - 2026-06-22

### 新增
- **CLAHE 自适应对比度增强** — 8×8 瓦片直方图均衡化 + 双线性插值，解决纸面光照不均
- **背景归一化（Retinex）** — 中值滤波估算背景 + 逐像素比值校正，消除阴影/折痕干扰
- **方向边缘增强（Sobel）** — X/Y 梯度幅值叠加原图，增强被噪声淹没的细笔画

### 优化
- 预处理策略从 12 种扩展至 15 种，投票覆盖面更广

## [2.4.0] - 2026-06-22

### 新增
- 确认页面显示字符置信度，低置信度（<60%）红色标记
- 每个字符卡片下方显示置信度百分比和彩色圆点
- 置信度颜色：绿（>=85%）、橙（60-85%）、红（<60%）
- 统计栏新增低置信度字符计数

### 优化
- 识别服务传递每字符置信度到 UI 层

## [2.3.0] - 2026-06-22

### 新增
- **投票提前终止** — 某字符票数过半时自动停止识别，速度提升 50%
- **形近字区分表** — 156 组形近字对照（如 已/己/巳、未/末/来），降低误识别
- **字典后处理优化** — 合并重复键，消除 const Map 编译错误

### 优化
- 修复 `voteMap`/`confidenceMap` 作用域错误，投票变量提前到第一轮之前定义
- 修复字典服务中 5 个重复键（无、来、目、禾、贝）
- 版本号修正为顺序递增（v2.2.0 → v2.3.0），不再跳号

## [2.2.0] - 2026-06-21

### 新增
- **首页 UI 重构** — 从堆砌列表改为分组网格布局，功能入口清晰分类
- **字体测试套件** — 内置测试文本（常用字、 pangram、标点），一键测试字体效果
- **撤销/重做系统** — 50 步操作历史，支持手势撤销
- **字体打包导出** — 一键打包 TTF + 预览图 + License 文件
- **Google Fonts 导出** — 生成 WOFF2 + CSS，可直接嵌入网页
- **批量照片导入** — 一次导入多张手写照片，批量识别
- **字体对比工具** — 并排对比两个字体的字形差异
- **自定义文本预览** — 输入任意文本实时预览字体效果
- **字符集完整性分析** — 检查常用字覆盖率，找出缺失字符
- **字形自动补全** — 基于已有字形风格推断缺失字符
- **字形评分系统** — 自动评估每个字形的质量分数
- **Kerning 编辑器** — 可视化调整字符间距
- **风格分析器** — 分析字体的笔画特征、倾斜度、连笔风格

### 优化
- 首页功能入口超过 20 个，全部重新分组（基础、编辑、分析、导出）
- 修复 Google Fonts 导出路由错误
- 修复 LateInitializationError in charset_screen

## [2.1.0] - 2026-06-21

### 新增
- **手写识别率优化** — 多级预处理投票机制，12 种预处理策略 × 多级放大
- **手写体专用预处理** — 笔画增强、倾斜校正、笔画归一化
- **云端识别 prompt 优化** — 更精准的指令提升云端识别准确率
- **本地+云端双重验证** — 本地 ML Kit 识别 + 云端 LLM 交叉验证
- **用户反馈学习** — 用户纠正的字符自动存储，下次识别优先匹配
- **笔画特征分析** — 笔画数量、方向、结构多维度辅助识别
- **多候选 Top-3** — 识别不确定时给出前 3 个候选
- **图像质量评估** — 自动检测模糊、低对比度、噪点并增强
- **OTF 导出** — 支持 OpenType 格式导出

### 修复
- 修复快速尝试跳过投票的 bug — 快速尝试结果改为计入投票
- 修复 30 处 setState after await 崩溃 — 添加 mounted 检查
- 修复版本号不一致问题
